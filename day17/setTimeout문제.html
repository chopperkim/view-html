색을 지정하는 방법

  1. #RRGGBB
  2. 색이름을 영문으로 쓴다(ex. black, pink, yellow)
색 만들기
0부터 255까지의 수를 16진수로 표현하면 된다.
255를 16진수로 표현하는 방법
var a = 255;
a.toString(16); // 왼쪽과 같이 하면 ff라는 문자가 출력된다.
만약에 빨간 색을 만들고 싶으면
var red = 255;
var green = 0;
var blue = 0;
document.body.style.backgroundColor = `#${red.toString(16)}${green.toString(16)}${blue.toString(16)}`;
